TestVerse — Nova

TestVerse is a hybrid AI-powered web automation testing platform. It combines a custom-trained Mini Transformer LLM with a locally-hosted Ollama model to deliver intelligent test script generation, natural language testing, visual regression detection, and real-time AI bug analysis — all running 100% locally on your machine.

Key Innovations
Hybrid AI Engine

Routes queries intelligently between a lightweight custom-trained PyTorch Mini Transformer for instant domain Q&A, and a full Ollama LLM for deep DOM analysis and script generation.

Visual Regression Testing

Captures and pixel-diffs browser screenshots at each test step using Pillow ImageChops. Detects UI regressions automatically with a 0.1% change threshold.

Natural Language → Test Script

Users describe tests in plain English. The Hybrid AI analyzes the live DOM and generates ready-to-run Selenium commands using a custom 3-command DSL.

100% Local & Private

No API keys, no cloud. All AI inference runs locally via Ollama, ensuring sensitive test data never leaves your machine.
